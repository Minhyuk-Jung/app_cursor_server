# ADR-006: 단일 VPS/온프렘 우선

> 상태: 확정 · 날짜: 2026-07-04

## 컨텍스트
배포 대상은 사용자 통제 VPS 또는 사내 온프렘. 초기 팀 규모·복잡도는 작다.

## 결정
- **단일 서버** 프로세스로 시작한다(스케줄러·RunEventLog·SessionManager in-process).
- 다중 서버·분산 큐는 **범위 외**(P7+ 검토, R-17 수용).

## 근거
- P0~P4 목표(1~수십 프로젝트, maxConcurrent 소규모)에 충분.
- 운영·디버깅 단순.

## 대안
- **Kubernetes 초기부터**: 과도한 운영 부담.
- **Cursor Cloud 호스팅**: 셀프호스팅 목표와 불일치.

## 결과
16 단일 노드 배포. NFR-14 99.5% 단일 SPOF 전제.
