# ADR-007: SDK·터미널 샌드박스 격리 (단계적)

> 상태: 확정(단계적) · 날짜: 2026-07-04

## 컨텍스트
터미널(13)은 컨테이너 격리를 계획했으나, SDK Local(04)은 초기에 호스트 cwd에서 실행되어 **AI 툴이 더 위험한 무격리 경로**가 된다(R-01).

## 결정
1. **목표**: SDK 에이전트와 사용자 터미널 모두 **동일 프로젝트 샌드박스**에서 실행.
2. **단계적**: P6 전까지는 완화 모드(승인·스코프·경로 제한·전용 OS 사용자)로 운영하고 문서화.
3. P6에서 샌드박스 런타임과 cwd 마운트 통합.

## 근거
- 사용자 터미널만 격리하는 비대칭은 보안 모델 역전.
- 컨테이너 없이 MVP는 가능하나 R-01 명시적 수용 필요.

## 대안
- **호스트 무격리 유지**: R-01 Critical 지속.
- **Cloud로 에이전트만 이전**: Local 파일·git 일관성 깨짐.
